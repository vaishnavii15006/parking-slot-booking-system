# Parking Slot Booking System

This project was developed as part of the Application Development course. The system helps users check parking slot availability, reserve parking spaces, manage bookings, and make payments through a simple web interface.

The application provides separate functionalities for users and administrators, making parking management more organized and efficient.

## Features

### User Module
- Register and login
- View available parking slots
- Book parking slots
- Cancel bookings
- View booking history
- Receive booking status updates

### Admin Module
- Manage parking slots
- Monitor bookings
- Track slot availability
- View parking records
- Manage parking operations through a dashboard

### Additional Features
- Real-time slot availability
- Booking cost calculation
- Secure user authentication
- REST API integration
- Responsive user interface

## Tech Stack

### Frontend
- React.js
- Material UI
- Axios

### Backend
- Spring Boot
- Spring Data JPA
- Hibernate
- REST APIs

### Database
- MySQL
